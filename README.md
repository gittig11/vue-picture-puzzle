# vue-picture-puzzle

## 简介

基于vue.js实现的 4*4 拼图小游戏。[点击预览](https://gittig11.github.io/vue-picture-puzzle/) 

技术栈：
```
vue
vue-router
vuex
sass
babel
eslint
```
